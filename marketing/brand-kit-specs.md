# AIIncomeLab - Brand Kit Specifications

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Primary BG | #0d1117 | Dark mode backgrounds |
| Secondary BG | #161b22 | Card backgrounds, headers |
| Accent | #58a6ff | Links, CTAs, highlights |
| Text Primary | #c9d1d9 | Body copy |
| Text Bright | #f0f6fc | Headlines |
| Success | #3fb950 | Income stats, positive metrics |
| Warning | #d29922 | Caution flags, "results vary" |
| Border | #30363d | Dividers, card borders |

## Typography

### Family
- **Primary:** Inter (headlines, body)
- **Monospace:** JetBrains Mono (code snippets, stats)
- **Fallback:** system-ui, sans-serif

### Sizes (for social graphics)
| Element | Size | Weight |
|---------|------|--------|
| Headline | 48-60px | 700 |
| Subhead | 32-40px | 600 |
| Body | 20-24px | 400 |
| Caption | 14-16px | 400 |
| Stat number | 64-80px | 800 |

## Logo

### Logo Mark
- Single letter "A" with AI-inspired bracket/styling
- Or full "AIIncomeLab" wordmark
- Always white/light on dark bg

### Logo Usage
- Minimum clear space: 1x height of letter on all sides
- Never place on busy backgrounds
- Never recolor outside brand palette

## Canva Templates to Create

### 1. Twitter Thread Cover
- 1200x675px
- BG: #0d1117 with subtle gradient
- Top: Thread number (e.g., "1/12")
- Center: Bold headline (48px Inter)
- Bottom: Handle @AIIncomeLab
- Accent underline in #58a6ff

### 2. LinkedIn Post Image
- 1200x627px
- Same structure, no thread number
- Include pull-quote style stat or insight

### 3. Instagram Carousel (per slide)
- 1080x1080px
- Slide 1: Title slide (hook + badge)
- Slides 2-4: Content slides (bullet points, stats)
- Slide 5: CTA slide ("Full guide at link in bio")

### 4. Instagram Reel Cover
- 1080x1920px
- Bold text overlay 60px
- AIIncomeLab watermark top-left

### 5. Pinterest Pin (vertical)
- 1000x1500px
- Keyword-rich title in bold
- Blog post URL visible
- Rich description in alt text

### 6. Story Template (IG Stories)
- 1080x1920px
- Q&A template, tip template, stat template
