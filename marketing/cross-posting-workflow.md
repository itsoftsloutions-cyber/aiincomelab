# AIIncomeLab - Cross-Posting Workflow

## Core Principle
One blog post → 4 platform-optimized repurposed formats.

---

## Workflow Steps

**Phase 1: Blog publishes**
- Wait for post to go live
- Extract: headline, 3-5 key stats/insights, 1 quotable line, 1 actionable tip

**Phase 2: Create platform assets (within 24h)**

| Asset | Platform | Time |
|-------|----------|------|
| Twitter/X thread | X | 20 min |
| LinkedIn post (long-form) | LinkedIn | 15 min |
| Instagram carousel (5 slides) | IG | 30 min |
| Pinterest pin | Pinterest | 10 min |

**Phase 3: Post within 48h of blog publish**

---

## Asset Recipes

### Twitter/X Thread (10-15 tweets)
```
1/ Hook tweet — bold claim or surprising stat from the post
2-9/ Value tweets — key insights, numbered steps, quotes
10/ CTA tweet — "Full breakdown in my latest post [link]"
     + 2-3 hashtags (#AITools #MakeMoneyOnline)
```
Scheduling: Typefully or manual

### LinkedIn Post (300-500 words)
```
- Hook (1-2 sentences, bold)
- Problem statement
- The approach (3-5 bullet points)
- Result/data point
- CTA to blog with summary
- 3-5 relevant hashtags
```
Scheduling: Native LinkedIn scheduler

### Instagram Carousel (5 slides)
```
Slide 1: Hook/Title slide — bold text, blog name
Slide 2: Key insight / stat
Slide 3: Step-by-step (3 steps)
Slide 4: Result or comparison
Slide 5: CTA — "Full guide at link in bio"
```
Design: Canva template (brand kit)
Scheduling: Later or Meta Business Suite

### Pinterest Pin
```
- Vertical image (1000x1500px)
- Title overlay: keyword-rich
- Pin description: 2-3 sentences with keywords
- Link: blog post with UTM
```
Scheduling: Native Pinterest scheduler

## Recommended Tool Stack

| Tool | Purpose | Cost |
|------|---------|------|
| Canva | All visual design | Free tier |
| Buffer | Cross-platform scheduling (3 channels) | Free tier |
| Later | IG scheduling + analytics | Free tier |
| Typefully | Twitter thread scheduling | Free tier |
| UTM.io | UTM link builder | Free |

### Phase 1 (MVP): Manual posting for first 2 weeks
Post manually to all platforms to validate strategy before committing to scheduling tools.

### Phase 2 (Week 3+): Automated scheduling
- Buffer for X + LinkedIn + Pinterest
- Later for Instagram
- Review after 30 days

## UTM Parameters

Every outgoing link must include:
```
?utm_source={platform}
&utm_medium=social
&utm_campaign={post-slug}
```

| Platform | utm_source value |
|----------|------------------|
| Twitter/X | twitter |
| LinkedIn | linkedin |
| Instagram | instagram |
| Pinterest | pinterest |
| Reddit | reddit |
| Quora | quora |
